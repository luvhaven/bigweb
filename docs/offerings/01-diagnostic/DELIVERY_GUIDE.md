## Phase 1: First Contact & Onboarding (0-2 hours)

### Step 1: Lead Intake
**When**: Immediately after form submission

**Actions**:
1. **Automated Response** (Immediate)
   - Send confirmation email with:
     - Thank you message
     - What to expect next
     - Timeline (48 hours)
     - Calendar link for optional kickoff call

2. **Manual Review** (Within 2 hours)
   - Review submitted website URL
   - Check if site is live and accessible
   - Note initial observations
   - Flag any technical issues

**Tools**:
- Email automation: Resend, SendGrid, or Supabase Edge Functions
- Calendar: Calendly or Cal.com
- CRM: Notion, Airtable, or HubSpot Free

### Step 2: Initial Assessment
**Duration**: 30 minutes

**Checklist**:
- [ ] Visit website on desktop
- [ ] Visit website on mobile
- [ ] Check page load speed (GTmetrix, PageSpeed Insights)
- [ ] Review navigation structure
- [ ] Identify primary conversion goals
- [ ] Screenshot key pages

**Tools**:
- **Speed Testing**: GTmetrix, PageSpeed Insights, WebPageTest
- **Screenshots**: Awesome Screenshot, Full Page Screen Capture
- **Mobile Testing**: BrowserStack, LambdaTest (free tier)

## Phase 2: Forensic Analysis (2-8 hours)

### Step 3: Technical Audit
**Duration**: 1-2 hours

**Analysis Points**:
1. **Performance**
   - Page load time (target: <3 seconds)
   - Time to Interactive (TTI)
   - Largest Contentful Paint (LCP)
   - Cumulative Layout Shift (CLS)

2. **Mobile Responsiveness**
   - Layout breaks
   - Touch target sizes
   - Font readability
   - Form usability

3. **SEO Basics**
   - Meta titles and descriptions
   - H1 tags
   - Image alt text
   - Internal linking

**Tools**:
- **Performance**: Lighthouse, WebPageTest, GTmetrix
- **Mobile**: Google Mobile-Friendly Test, BrowserStack
- **SEO**: Screaming Frog (free 500 URLs), Ahrefs Webmaster Tools (free)
- **Accessibility**: WAVE, axe DevTools

### Step 4: UX Friction Analysis
**Duration**: 2-3 hours

**Framework**: Use the "5-Second Test" method

**Analysis Areas**:
1. **Homepage**
   - [ ] Is the value proposition clear in 5 seconds?
   - [ ] Is the primary CTA obvious?
   - [ ] Is there a clear next step?
   - [ ] Are there trust signals (testimonials, logos)?

2. **Navigation**
   - [ ] Can users find what they need in 2 clicks?
   - [ ] Is the menu structure logical?
   - [ ] Are CTAs consistent?

3. **Forms**
   - [ ] How many fields? (Fewer = better)
   - [ ] Is there inline validation?
   - [ ] Is the submit button prominent?
   - [ ] Is there a privacy statement?

4. **Conversion Funnel**
   - Map the user journey from landing to conversion
   - Identify drop-off points
   - Note friction elements (pop-ups, redirects, etc.)

**Tools**:
- **Heatmaps**: Hotjar (free plan), Microsoft Clarity (free)
- **Session Recording**: Hotjar, FullStory, Microsoft Clarity
- **User Testing**: UserTesting.com, Maze, UsabilityHub
- **Analytics**: Google Analytics 4, Plausible
