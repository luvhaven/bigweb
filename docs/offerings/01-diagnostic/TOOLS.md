# Tools & Resources for Conversion Diagnostic

## Category 1: Performance & Technical Analysis

### Speed Testing
1. **GTmetrix** (Free + Paid)
   - URL: https://gtmetrix.com
   - Use: Comprehensive page speed analysis
   - Cost: Free (limited), $10/mo (Pro)

2. **PageSpeed Insights** (Free)
   - URL: https://pagespeed.web.dev
   - Use: Google's official speed tool
   - Cost: Free

3. **WebPageTest** (Free)
   - URL: https://www.webpagetest.org
   - Use: Advanced performance testing
   - Cost: Free

### Mobile Testing
1. **BrowserStack** (Paid)
   - URL: https://www.browserstack.com
   - Use: Test on real devices
   - Cost: $29/mo (starts)

2. **LambdaTest** (Free + Paid)
   - URL: https://www.lambdatest.com
   - Use: Cross-browser testing
   - Cost: Free tier available
