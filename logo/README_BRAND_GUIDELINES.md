# BIGWEB Digital — Brand Logo Suite

This directory contains the master vector formats for the BIGWEB Digital logo, created for varying applications including web, print, apparel, and large format media.

All files are provided in infinitely scalable `SVG` format. If you need EPS, PDF, or PNG formats for a specific printer, any modern vector software (Adobe Illustrator, Figma, Affinity Designer) can export them directly from these SVGs with zero quality loss.

## The Logo Suite

### 1. Logomark (The Geometric Grid)
- **`01_logomark_primary.svg`**: The core geometric brand mark in the official Gold (`#D4AF6A`). Use as a standalone avatar, watermark, or favicon.
- **`02_logomark_monochrome_black.svg`**: Solid black version. Use for 1-color screen printing (shirts, tote bags), rubber stamps, or embossing.
- **`03_logomark_monochrome_white.svg`**: Solid white version. Use for 1-color printing on dark apparel or surfaces.

### 2. Horizontal Layouts (Standard)
The default layout for most digital and print applications where vertical space is constrained (e.g., website headers, letterheads, email signatures).
- **`04_logo_horizontal_dark_bg.svg`**: Features white text. Optimized for use on dark backgrounds (`#080706` or similar). *(Note: Includes a dark background container for previewing; delete the background rectangle layer before placing on your own dark backgrounds if needed).*
- **`05_logo_horizontal_light_bg.svg`**: Features dark charcoal text. Optimized for use on white paper, light backgrounds, and standard print documents.

### 3. Stacked Layouts (Large Format)
The "Badge" layout. Use this when the logo needs to be the hero element, or when horizontal space is constrained (e.g., posters, square social media posts, apparel centers, sponsor grids).
- **`06_logo_stacked_dark_bg.svg`**: Centered mark above white text.
- **`07_logo_stacked_light_bg.svg`**: Centered mark above dark text.

## Typography Notes
If editing the wordmark, ensure you have the correct fonts installed on your system:
- **BIGWEB**: `Playfair Display` (BIG is Black/900 weight, WEB is Regular/400 weight).
- **Digital**: `DM Sans` (Medium/500 weight, all caps, wide tracking).

## Brand Colors
- **Gold**: `#D4AF6A` (CMYK: `15, 25, 60, 5`)
- **Dark Charcoal**: `#080706` (CMYK: `75, 68, 67, 90`)
- **White**: `#FFFFFF` (CMYK: `0, 0, 0, 0`)
